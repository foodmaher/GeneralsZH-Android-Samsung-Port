# Development Diary - July 2026

## 2026-07-22: Add the Xclipse RGBA8 texture proof

Added an opt-in compatibility path for Android Vulkan drivers that cannot
sample the game's BC1/BC2/BC3 textures. The proof converts legally extracted
DDS files to classic A8R8G8B8 offline, installs them under an isolated loose
override root, and leaves the default Adreno/Turnip path alone.

The engine now reads classic A8R8G8B8 DDS mip chains through the existing W3D
texture loader. Android startup and DXVK logs expose the physical device,
driver, BC capability, selected profile, selected game-data directory, and
whether libadrenotools is active. Synthetic Python tests cover DXT1 alpha,
DXT3, DXT5, mip chains, odd dimensions, bad headers, and incremental output.

RGBA8 is deliberately the first correctness proof. It costs roughly 8× BC1 or
4× BC2/BC3 storage and GPU memory; ASTC/ETC2 remain later optimisation work.
